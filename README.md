# Chess Game (SFML)

A simple 2D Chess game built using *C++* and *SFML*. The game allows two players to play against each other on the same computer. Features include basic chess rules, move validation, pawn promotion, castling, en-passant, and capture tracking.

---

## *Features*

- Fully functional chessboard (8x8)
- Move validation for all pieces (Pawn, Bishop, Knight, Rook, Queen, King)
- Castling and En-Passant implemented
- Pawn promotion
- Highlights legal moves for selected pieces
- Capture tracking
- Turn-based gameplay (White/Black)
- Simple GUI using *SFML*

---
